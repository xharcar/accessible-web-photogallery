This is a Master's thesis project titled "Accessible web photo gallery". It aims to provide a simple, accessible,
easy-to-use photo gallery for everyone, including visually impaired people. It's capable of processing JSON
metadata about buildings within the camera's FOV, as created by the AssistiveCamera application, developed by
Ondřej Lapáček at the Faculty of Informatics of Masaryk University, Brno, Czechia.
The source code is available under GPL 3.0 on GitHub.
It uses Nominatim API functionality.
Credit for all building data goes to OpenStreetMap contributors.
